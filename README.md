# manageiq-exchange-frontend
React application that integrates with the manageiq-EXCHANGE backend

# Configure app
  -  EXCHANGE_API_BACKEND:
  -  EXCHANGE_API_VERSION:
  -  GITHUB_OAUTH_ID:

## Start & watch

    $ npm run start:react
    $ npm run start-dev-hmr
## Simple build for production

    $ npm run build
## Test
    $ npm run test
